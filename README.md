# RandomFunctionsEvolutionVisualization

![alt text](https://github.com/k1s4g4/RandomFunctionsEvolutionVisualization/blob/master/screenshot/4.png)

This program was made as a way of better understanding how random functions work. Our understanding is better when we see things instead of just imagining them.It just generates random values within an interval, calculates relative frequencies (probability) for each value and plots probability distributions for six different functions.
